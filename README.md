# 🧩 알고리즘 스터디

팀원들과 함께 알고리즘 문제를 풀이하고 기록하는 저장소입니다.

## 🎯 목표

- 코딩테스트 대비 문제 해결 능력 향상시킵니다.
- 다양한 알고리즘 유형을 학습합니다.
- 하루에 최소 1문제씩 풀며 꾸준한 문제 풀이 습관을 형성합니다.

## 🚀 진행 방식

- 스터디는 주 단위로 진행하며, 모임 전날까지 정해진 문제를 각자 풀이 후 GitHub에 업로드합니다.
- 코드 리뷰 및 풀이 공유를 통해 다양한 접근 방식을 학습합니다.
- 스터디 진행 시의 문제는 미리 정해놓지만, 각자 공유하고 싶은 문제를 추가하는것도 환영입니다!
- 추후 난이도 상승 시 합의 후 코드래빗을 도입 고려 중입니다.

## 📏 규칙

- 주어진 기간 엄수!
- 스터디 진행 시 최소 1문제 이상 풀이하기
- 참여를 못하는 경우에는 전날에 미리 말해주기

### 🗣️ 언어

- 파이썬, 자바, 자바스크립트 중 원하는 언어로 진행합니다.

### ⚠️ 커밋 컨벤션

- 형식: `[플랫폼명] 문제번호 문제이름 - 본인이름`

- 예시
  - `[BOJ] 1330 두 수 비교하기 - 박준형`
  - `[PGS] 두 수의 합 - 박준형`

### 📝 제출 방식

0. **본인 이름의 브랜치로 체크아웃**: `git checkout -b <MemberName>` (최초 생성 후에는 해당 브랜치를 쭉 사용합니다.)
1. **문제 풀이 코드 및 README 작성**
2. **Commit & Push**
3. **PR 생성**: PR 제목은 "[week01] 본인이름"으로 통일합니다.
4. **회의 종료 후 Merge**
